Perform Math Operations via URL

To perform a math operation using a URL, use the following format:

https://ourpublicip_or_domainname/cgi-bin/math.cgi?OPERATION=OPERANDS*OPERATION

Replace OPERATION with the desired math operation (ADD, SUBTRACT, MULTIPLY, DIVIDE), and replace EXPRESSION with the expression to evaluate.

Example on how to perform an addition:

https://ourpublicip_or_domainname/cgi-bin/math.cgi?ADD=1+1
